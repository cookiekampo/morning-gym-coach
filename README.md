# Morning Gym Coach

朝ジムで30〜45分の脚トレを、スマホに脱線せず完走するための個人用PWAです。iPhone Safariで開き、ホーム画面に追加して使います。

## v0.1機能

- 固定メニューを見る
- セット結果を入力する
- RIRを選ぶ
- 記録後に休憩タイマーを自動開始する
- 次セット指示を表示する
- localStorageにセッション記録を残す
- トレ後に予定と実績の差分を見る
- 直前セットの簡単な修正 / 削除
- 痛みありボタンによる安全停止

## 初期脚メニュー

脚 / 45分

| 種目 | 重量 | セット | 回数 | 目標RIR | 休憩 | 注意 |
| --- | ---: | ---: | --- | --- | ---: | --- |
| スクワット | 40kg | 3 | 6〜8回 | 3 → 2 → 1 | 180秒 | RIR0禁止 |
| ブルガリアンスクワット | 片手10kg | 2 | 片脚8〜10回 | 2 → 1 | 120秒 | RIR0禁止 |
| レッグエクステンション | 30kg | 2 | 12〜15回 | 1 → 0 | 60秒 | 最終セットのみRIR0 / レストポーズ可 |

ブルガリアンスクワットの `planned_weight_kg` は片手あたりのダンベル重量です。左右それぞれ同じ回数を行う前提です。

## RIR / RPEルール

RIRは「同じフォームであと何回できたか」です。反動、浅い可動域、フォーム崩れ込みの回数は含めません。

| RIR | RPE |
| --- | ---: |
| 4+ | 6 |
| 3 | 7 |
| 2 | 8 |
| 1 | 9 |
| 0 | 10 |

## 安全ルール

- スクワットはRIR0禁止
- ブルガリアンスクワットはRIR0禁止
- レッグエクステンションの最終セットのみRIR0可
- レッグエクステンションのレストポーズは最終セットのみ可
- `all_out_allowed: false` の種目でRIR0を選んだ場合はRIR1に補正
- 痛みありを押したら、その種目を終了
- 痛みあり後は、その日のオールアウト禁止
- 痛みあり後は、レッグエクステンションでもRIR0とレストポーズ禁止
- 予定重量 `plannedWeightKg` と実績重量 `actualWeightKg` は分けて保存し、予定は書き換えない

## ローカル確認方法

`docs/index.html` を直接開かず、ローカルサーバーで確認します。

```bash
python -m http.server 8000 --directory docs
```

ブラウザで以下を開きます。

```text
http://localhost:8000
```

## GitHub Pages公開方法

1. GitHubで新規リポジトリ `morning-gym-coach` を作る
2. このフォルダをpushする
3. Settings → Pages を開く
4. Source を Deploy from a branch にする
5. Branch を `main`、folder を `/docs` にする
6. Saveする
7. 公開URLは `https://<username>.github.io/morning-gym-coach/`
8. URLに `/docs/` は付けない

## iPhoneで使う手順

1. iPhoneのSafariで公開URLを開く
2. 共有ボタンを押す
3. 「ホーム画面に追加」を押す
4. ホーム画面から Morning Gym Coach を開く
