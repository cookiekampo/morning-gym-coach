# Morning Gym Coach

朝ジムで30〜45分の脚・胸・背中トレを、スマホに脱線せず完走するための個人用PWAです。iPhone Safariで開き、ホーム画面に追加して使います。

## v0.1機能

- 固定メニューを見る
- セット結果を入力する
- RIRを選ぶ
- 記録後に休憩タイマーを自動開始する
- 次セット指示を表示する
- localStorageにセッション記録を残す
- トレ後に予定と実績の差分を見る
- 直前セットの簡単な修正 / 削除
- 痛みありボタンによる安全停止

## RIR / RPEルール

RIRは「同じフォームであと何回できたか」です。反動、浅い可動域、フォーム崩れ込みの回数は含めません。

| RIR | RPE |
| --- | ---: |
| 4+ | 6 |
| 3 | 7 |
| 2 | 8 |
| 1 | 9 |
| 0 | 10 |

## 安全ルール

- スクワットのRIR0は非推奨だが、警告確認後に記録可能
- ブルガリアンスクワットのRIR0も非推奨だが、警告確認後に記録可能
- レッグエクステンションはRIR0を記録可能
- レッグエクステンションのレストポーズは最終セットのみ可
- ペクトラルフライは胸コースの基本オールアウト種目
- ペクトラルフライのRIR0とレストポーズは最終セットのみ可
- デッドリフトのRIR0は非推奨だが、警告確認後に記録可能
- ラットプルダウンとシーテッドローのRIR0も非推奨だが、警告確認後に記録可能
- マシンロウイングは背中コースの基本オールアウト種目
- マシンロウイングのRIR0とレストポーズは最終セットのみ可
- RIR0は自動でRIR1に補正しない
- 痛みありを押したら、その種目を終了
- 痛みあり後は、その日のオールアウト禁止
- 痛みあり後は、レッグエクステンション、ペクトラルフライ、マシンロウイングでもRIR0とレストポーズ禁止
- 予定重量 `plannedWeightKg` と実績重量 `actualWeightKg` は分けて保存し、予定は書き換えない

## 通知について

- サーバープッシュ通知は使いません
- Push APIは使いません
- 外部サーバーは使いません
- アプリ起動中のタイマー終了通知として、画面表示・音・バイブ・Notification APIを使います
- Notification APIは対応環境で、ユーザーが許可した場合のみ使います

## ローカル確認方法

`docs/index.html` を直接開かず、ローカルサーバーで確認します。

```bash
python -m http.server 8000 --directory docs
```

ブラウザで以下を開きます。

```text
http://localhost:8000
```

## GitHub Pages公開方法

1. GitHubで新規リポジトリ `morning-gym-coach` を作る
2. このフォルダをpushする
3. Settings → Pages を開く
4. Source を Deploy from a branch にする
5. Branch を `main`、folder を `/docs` にする
6. Saveする
7. 公開URLは `https://<username>.github.io/morning-gym-coach/`
8. URLに `/docs/` は付けない

## iPhoneで使う手順

1. iPhoneのSafariで公開URLを開く
2. 共有ボタンを押す
3. 「ホーム画面に追加」を押す
4. ホーム画面から Morning Gym Coach を開く
