# Morning Gym Coach

朝ジムで30〜45分の脚トレを、スマホに脱線せず完走するための個人用PWAです。iPhone Safariで開き、ホーム画面に追加して使います。

## v0.1機能

- 固定メニューを見る
- セット結果を入力する
- RIRを選ぶ
- 記録後に休憩タイマーを自動開始する
- 次セット指示を表示する
- localStorageにセッション記録を残す
- トレ後に予定と実績の差分を見る
- 直前セットの簡単な修正 / 削除
- 痛みありボタンによる安全停止

## v0.2の変更点

- 記録削除機能を追加
- 規定セット後のセット追加機能を追加
- RIR0は警告付きで全種目記録可能
- RIR0は自動補正せず、RPE10として保存
- 種目切り替えボタンを「次セットへ」「次の種目へ：種目名」「トレーニング終了・まとめへ」に改善
- タイマー終了時に画面上で「休憩終了」を表示
- 対応環境ではタイマー終了時に音またはバイブを実行
- Notification APIは対応環境のみ使用
- 通知許可は「通知を許可」ボタンを押した時だけ要求
- サーバープッシュ通知は未実装
- GitHub Pagesは `main / docs` 公開前提

## 初期脚メニュー

脚 / 45分

| 種目 | 重量 | セット | 回数 | 目標RIR | 休憩 | 注意 |
| --- | ---: | ---: | --- | --- | ---: | --- |
| スクワット | 40kg | 3 | 6〜8回 | 3 → 2 → 1 | 180秒 | RIR0は警告付き |
| ブルガリアンスクワット | 片手10kg | 2 | 片脚8〜10回 | 2 → 1 | 120秒 | RIR0は警告付き |
| レッグエクステンション | 30kg | 2 | 12〜15回 | 1 → 0 | 60秒 | RIR0可 / 最終セットのみレストポーズ可 |

ブルガリアンスクワットの `planned_weight_kg` は片手あたりのダンベル重量です。左右それぞれ同じ回数を行う前提です。

## RIR / RPEルール

RIRは「同じフォームであと何回できたか」です。反動、浅い可動域、フォーム崩れ込みの回数は含めません。

| RIR | RPE |
| --- | ---: |
| 4+ | 6 |
| 3 | 7 |
| 2 | 8 |
| 1 | 9 |
| 0 | 10 |

## 安全ルール

- スクワットのRIR0は非推奨だが、警告確認後に記録可能
- ブルガリアンスクワットのRIR0も非推奨だが、警告確認後に記録可能
- レッグエクステンションはRIR0を記録可能
- レッグエクステンションのレストポーズは最終セットのみ可
- RIR0は自動でRIR1に補正しない
- 痛みありを押したら、その種目を終了
- 痛みあり後は、その日のオールアウト禁止
- 痛みあり後は、レッグエクステンションでもRIR0とレストポーズ禁止
- 予定重量 `plannedWeightKg` と実績重量 `actualWeightKg` は分けて保存し、予定は書き換えない

## 通知について

- サーバープッシュ通知は使いません
- Push APIは使いません
- 外部サーバーは使いません
- アプリ起動中のタイマー終了通知として、画面表示・音・バイブ・Notification APIを使います
- Notification APIは対応環境で、ユーザーが許可した場合のみ使います

## ローカル確認方法

`docs/index.html` を直接開かず、ローカルサーバーで確認します。

```bash
python -m http.server 8000 --directory docs
```

ブラウザで以下を開きます。

```text
http://localhost:8000
```

## GitHub Pages公開方法

1. GitHubで新規リポジトリ `morning-gym-coach` を作る
2. このフォルダをpushする
3. Settings → Pages を開く
4. Source を Deploy from a branch にする
5. Branch を `main`、folder を `/docs` にする
6. Saveする
7. 公開URLは `https://<username>.github.io/morning-gym-coach/`
8. URLに `/docs/` は付けない

## iPhoneで使う手順

1. iPhoneのSafariで公開URLを開く
2. 共有ボタンを押す
3. 「ホーム画面に追加」を押す
4. ホーム画面から Morning Gym Coach を開く
